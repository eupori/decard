개인정보처리방침

데카드(Decard)

1. 개인정보의 수집 항목 및 방법

데카드는 서비스 제공을 위해 아래와 같은 개인정보를 수집합니다.

수집 경로 수집 항목
카카오 로그인 닉네임, 프로필 이미지
비로그인 이용 디바이스 식별자(Device ID)
서비스 이용 카드 생성·학습·복습 이력

2. 개인정보의 수집·이용 목적

  • 회원 식별 및 서비스 제공
  • 학습 기록 관리 및 개인화된 학습 경험 제공
  • 서비스 개선 및 신규 기능 개발
  • 서비스 이용 통계 분석

3. PDF 파일 처리 정책

이용자가 업로드한 PDF 파일은 텍스트 추출 후 암기카드 생성에만 사용되며, 원본 PDF 파일은 서버에 저장하지 않습니다.

추출된 텍스트는 카드 생성 과정에서 AI(Anthropic Claude API)로 전송됩니다. 카드 생성이 완료되면 추출 텍스트도 보관하지 않습니다.

4. 개인정보의 보유 및 이용 기간

이용자의 개인정보는 수집·이용 목적이 달성된 후 지체 없이 파기합니다.

  • 회원 탈퇴 요청 시: 즉시 파기
  • 서비스 종료 시: 종료일로부터 30일 이내 파기
  • 비로그인 이용자의 디바이스 ID 및 학습 기록: 마지막 이용일로부터 1년 경과 시 파기

5. 개인정보의 제3자 제공

데카드는 이용자의 개인정보를 원칙적으로 제3자에게 제공하지 않습니다. 다만, 아래의 경우는 예외로 합니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 의해 요구되는 경우
암기카드 생성을 위해 PDF에서 추출된 텍스트가 Anthropic사의 Claude API로 전송됩니다. 해당 데이터는 카드 생성 목적으로만 사용되며, Anthropic의 API 이용약관에 따라 처리됩니다.
