# 데카드(Decard) 테스터 가이드

## 앱 소개
PDF를 올리면 AI가 근거 포함 암기카드를 자동 생성해주는 시험 대비 학습 도구입니다.

---

## 다운로드

| 플랫폼 | 링크 |
|--------|------|
| 웹 (PC/모바일) | https://decard.eupori.dev |
| Android APK | [GitHub Releases](https://github.com/cml-eupori/decard/releases/latest) |

> iOS는 현재 미지원입니다.

---

## 사용법

### 1단계: PDF 업로드
- 웹: 홈 화면에서 "PDF 업로드" 버튼 클릭
- 앱: 홈 화면에서 "PDF 업로드" 버튼 탭
- 텍스트 기반 PDF만 지원 (스캔본 미지원)
- 파일 크기 10MB / 100페이지 이하

### 2단계: 카드 확인 및 검수
- AI가 자동 생성한 카드를 확인합니다
- 카드별로 채택/삭제/수정 가능
- "전체 채택" 버튼으로 한번에 채택 가능

### 3단계: 학습
- 채택한 카드로 플래시카드 학습
- 카드를 넘기며 암기 여부 확인
- CSV 다운로드로 Anki 등 외부 도구 연동 가능
