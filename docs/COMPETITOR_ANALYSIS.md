> 작성일: 2026-02-26

## 1. Anki 심층 분석

### 1.1 개요

| 항목 | 내용 |
|------|------|
| **정식명** | Anki (暗記, 일본어 "암기") |
| **개발자** | Damien Elmes (2006~2026.2), 현재 AnkiHub 팀 이관 중 |
| **유형** | 오픈소스 플래시카드 + 간격 반복(SRS) 학습 도구 |
| **라이선스** | AGPL-3.0 (오픈소스, 영구 유지 약속) |
| **최초 출시** | 2006년 10월 |
| **최신 버전** | Desktop 25.x / AnkiDroid 2.23.3 (2026.1 업데이트) |

### 1.2 세부 기능

#### 핵심 기능
- **간격 반복 학습 (SRS)**: SM-2 기반 + FSRS(차세대 알고리즘) 선택 가능
- **카드 유형**: 기본(앞/뒤), 빈칸(Cloze), Image Occlusion, 역방향 카드
- **노트-카드 분리 구조**: 하나의 노트에서 여러 카드 자동 생성 (예: 앞→뒤, 뒤→앞)
- **HTML/CSS/JS 카드 템플릿**: 완전한 커스터마이징 (LaTeX, MathJax, 미디어 포함)
- **덱 관리**: 계층형 덱, 태그, 플래그, 필터 덱, 커스텀 학습 세션
- **통계/분석**: 학습 그래프, 리뷰 수, 예측 보유율, 시간대별 성과
- **AnkiWeb 동기화**: 모든 플랫폼 간 무료 동기화

#### SRS 알고리즘

| 알고리즘 | 특징 |
|----------|------|
| **SM-2 (레거시)** | SuperMemo 기반, 모든 유저 동일 공식, 수동 파라미터 조정 |
| **FSRS (v23.10+)** | ML 기반, 7억 건 리뷰 데이터 학습, 개인화된 스케줄링, SM-2 대비 20~30% 리뷰 감소 |

FSRS는 "Three Component Model of Memory" 기반으로 Retrievability(기억 확률), Stability(안정성), Difficulty(난이도) 3가지 변수를 추적.

#### 플러그인/애드온 시스템
- **1,600개 이상** 애드온 (AnkiWeb에서 설치 가능)
- 인기 애드온: Image Occlusion Enhanced, Review Heatmap, Night Mode, Speed Focus Mode, Frozen Fields 등
- **데스크톱 전용** (모바일에서는 애드온 미지원)
- Python 기반 API로 자유로운 기능 확장

#### 공유 덱 커뮤니티
- AnkiWeb Shared Decks: 수천 개의 커뮤니티 제작 덱 무료 공유
- **AnkiHub**: 협업 덱 플랫폼 (의대생 STEP 1/2 덱 등 대규모 협업 관리)
- 의대 커뮤니티 중심으로 매우 활성화

#### 지원 플랫폼

| 플랫폼 | 앱명 | 비용 |
|--------|------|------|
| Windows/Mac/Linux | Anki Desktop | 무료 |
| Android | AnkiDroid | 무료 |
| iOS/iPadOS | AnkiMobile | $24.99 (일회성) |
| 웹 | AnkiWeb | 무료 (리뷰만, 편집 제한) |

**약점**: 극심한 러닝커브, 구시대적 UI/UX, 카드 생성 노동집약적, 모바일 애드온 불가, iOS $24.99, PDF→카드 네이티브 불가

## 2. Quizlet 심층 분석

### 2.1 개요

| 항목 | 내용 |
|------|------|
| **서비스명** | Quizlet |
| **설립** | 2005년 |
| **슬로건** | "AI-powered Flashcards" |
| **지원 플랫폼** | Web, iOS, Android |
| **지원 국가** | 130개국 이상 |
| **직원 수** | 547명 |

### 2.2 세부 기능

#### 핵심 학습 모드
- **Flashcards**: 기본 플래시카드 (Know / Still Learning 분류)
- **Learn**: AI 기반 적응형 학습 — 틀린 문제 반복, 개인화된 학습 경로
- **Test**: 객관식/참거짓/서술형 혼합 연습 시험 자동 생성
- **Match**: 게임화된 매칭 모드 (카드 짝 맞추기, 시간 기록)
- **Write**: 서술형 답변 연습

#### AI 기능
- **Magic Notes**: 필기/슬라이드/PDF 업로드 → 아웃라인, 플래시카드, 연습시험, 스터디 가이드 자동 생성
- **AI Flashcard Generator**: PDF, Word, Google Drive → 플래시카드 즉시 생성
- **AI Practice Tests**: AI가 자동으로 연습시험 생성
- **Expert Solutions**: 실제 교재의 단계별 풀이 제공
- **Memory Score**: 학습 정도를 수치화하여 추적
- **Scheduled Reviews**: 복습 일정 자동 설정 (망각 곡선 기반)

#### 콘텐츠 생태계
- **7억+ 플래시카드 세트** (사용자 생성)
- 텍스트→음성 변환 (TTS)
- 크로스 디바이스 동기화

### 2.3 고객 평가

| 플랫폼 | 평점 | 리뷰 수 |
|--------|------|---------|
| Google Play | 4.7 | 811,000+ |
| App Store | 4.7 | 수십만+ |
| Trustpilot | **1.4** | 500+ |

> 앱스토어에서는 높은 평점이지만, Trustpilot에서는 **유료화 정책 반발**로 극단적으로 낮음

**긍정 키워드**: "사용하기 쉽다", "다양한 학습 모드", "남의 세트를 바로 쓸 수 있다", "시험 전 빠른 복습에 최적", "게임화가 동기 부여"

**부정 키워드**: "Paywall (가장 빈번)", "광고가 너무 많다", "데이터 내보내기 불가 (락인)", "AI 답변 부정확 (~13%)", "모바일 앱 버그/로딩 느림"

### 2.4 이용자 수

| 지표 | 수치 |
|------|------|
| 누적 학습자 | **3억 명+** |
| 월간 활성 사용자 (MAU) | **5,000만+** |
| 플래시카드 세트 수 | **7억+** |
| 유료 구독자 | ~100만 명 |

### 2.5 수익 모델

- **2025년 매출**: $139M (전년 대비 73.7% 성장)
- **Quizlet Plus**: 월 $7.99 / 연 $35.99

| 기능 | 무료 | Plus |
|------|------|------|
| 플래시카드 생성/사용 | O | O |
| Learn / Test 모드 | 일일 제한 | 무제한 |
| Magic Notes (AI 카드생성) | 제한적 | O |
| AI Practice Tests | X | O |
| 광고 제거 | X | O |
| 오프라인 학습 | X | O |
| 이미지/오디오 첨부 | X | O |

### 2.6 강점과 약점

**강점**: 압도적 콘텐츠 생태계(7억+), 다양한 학습 모드, 게임화, 크로스 플랫폼, AI 통합, 브랜드 인지도 1위, 네트워크 효과

**약점**: 공격적 유료화(Trustpilot 1.4점), 광고 과다, 데이터 락인(내보내기 제거), AI 정확도 이슈, 한국 현지화 부족, PDF 근거 매핑 없음

## 3. Brainscape 심층 분석

### 3.1 개요

| 항목 | 내용 |
|------|------|
| **앱명** | Brainscape |
| **설립** | 2010년 (Andrew S. Cohen 창업) |
| **본사** | 뉴욕, 미국 |
| **직원** | ~42명 |
| **플랫폼** | iOS, Android, Web |
| **슬로건** | "The World's Smartest Flashcard App" |

### 3.2 세부 기능

#### 핵심 기능
- **CBR (Confidence-Based Repetition)**: 독자적 간격반복. 카드마다 1~5 자신감 등급 → 낮은 점수 카드 더 자주 노출
- **Certified Content**: 의학/법학/언어 전문 출판사와 협력한 공인 플래시카드 (유료)
- **AI 카드 생성**: PDF, PPT, DOC 업로드 → AI가 플래시카드 자동 생성 (Pro)
- **협업 학습**: 클래스/그룹 단위 덱 공유, 학생 진도 모니터링
- **멀티미디어**: 이미지, 오디오 첨부 (Pro)

#### 독특한 학습 방식
- **5단계 자신감 평가**: "전혀 모름(1)" ~ "완벽히 안다(5)"
- **구조화된 코스**: 전문가 설계 학습 코스 형태
- **Mixed Study Mode**: 여러 덱 섞어서 학습

### 3.3 고객 평가

| 플랫폼 | 평점 | 리뷰 수 |
|--------|------|---------|
| Google Play | 4.7 | ~13,000 |
| App Store | 4.8 | - |
| Trustpilot | **4.8** | 464 |

**긍정 키워드**: "과학적인 반복 학습", "직관적 UI", "광고 없음", "의대 공부에 최적", "Anki보다 쉽고 Quizlet보다 효과적"

**부정 키워드**: "무료 기능이 너무 제한적", "월 구독료가 비쌈", "카드 수동 생성 느림", "오프라인 기능 부족"

### 3.4 이용자 수

| 지표 | 수치 |
|------|------|
| 전체 등록 유저 | ~1,000만+ |
| Google Play 다운로드 | 100만+ |
| 주요 사용자층 | 의대생, 법대생, 자격증 준비생, 기업 교육 |

### 3.5 수익 모델

- **2024년 매출**: $6.1M (39% 성장, 42명 소규모 팀)

| 플랜 | 가격 |
|------|------|
| 월간 | $19.99/월 |
| 학기 | $9.99/월 |
| 연간 | $7.99/월 |
| 평생 | $199.99 (1회) |

### 3.6 강점과 약점

**강점**: 직관적 CBR, 공인 콘텐츠 생태계, 깔끔한 UI + 광고 없음, 기업 교육 B2B, 과학적 브랜딩

**약점**: 비싼 구독료(월 $20), 무료 사용자 제약, 커뮤니티 규모 작음, 커스터마이징 부족

## 5. 각 경쟁사의 핵심 약점 = Decard의 기회

| 경쟁사 | 치명적 약점 | Decard 기회 |
|---|---|---|
| **Anki** | 러닝커브 극심 + 카드 수동 생성 노동 + 구식 UI | "AI가 다 해주는 현대적 Anki" |
| **Quizlet** | 공격적 유료화 반발 + AI 부정확 + 데이터 락인 | "무료 AI 생성 + 데이터 자유(CSV 내보내기)" |
| **Brainscape** | 비싼 구독(월$20) + AI가 부가기능 수준 | "AI 생성이 핵심, 무료/저가" |

## 6. Decard만의 유일한 강점

### "PDF → 근거(출처 페이지) 포함 카드 자동 생성"

어떤 경쟁사도 제공하지 않는 기능:
- **Anki**: PDF→카드 기능 자체가 없음
- **Quizlet**: PDF→카드는 있지만 근거/페이지 매핑 없음
- **Brainscape**: PDF→카드는 있지만 유료 + 근거 없음

## 7. Decard 아이덴티티 전략

### 포지셔닝 한 줄

> **"PDF 올리면 끝. 근거까지 알려주는 AI 시험카드"**

### 패러다임 전환

| 기존 앱들의 패러다임 | Decard의 패러다임 |
|---|---|
| 사용자가 카드를 **만든다** | AI가 카드를 **만들어준다** |
| 카드 = 질문 + 답 | 카드 = 질문 + 답 + **왜 그 답인지 근거** |
| "이 답 맞아?" → 모름 | "이 답 맞아?" → **교재 p.23에 나와있음** |
| 학습 도구를 배워야 한다 | PDF만 올리면 된다 |

### 3단계 아이덴티티 로드맵

**1단계 (현재~6개월): "제로 노동 카드 생성"**
- 메인 킥: **"카드 만드는 시간 = 0"**
- 타겟: 시험 1~2주 전 급한 대학생
- 킬러 시나리오: "교수 PPT 200페이지 → 5분만에 50장 시험카드"
- Anki/Quizlet 대비 압도적 시간 절약이 훅

**2단계 (6~12개월): "이해 기반 학습"**
- 메인 킥: **"단순 암기가 아닌, 근거를 아는 학습"**
- 근거+페이지 포함 → "왜?"를 알며 공부 → 응용 문제도 대응
- SRS(간격 반복) 추가로 장기 기억 지원
- Anki 유저 중 "효율은 좋지만 이해는 안 되는" 불만층 흡수

**3단계 (12개월+): "한국 시험 대비 No.1"**
- 한국 교육과정/자격증 특화 (수능, 공무원, 의대, 법대)
- 대학교/학원 B2B (교수가 PDF 올리면 → 학생들에게 카드 배포)
- 커뮤니티 덱 공유 (같은 과목 수강생끼리)

### 슬로건 후보

| 후보 | 뉘앙스 |
|---|---|
| **"PDF 올리면 시험카드 완성"** | 기능 직접 설명 (명확) |
| **"카드는 AI가, 공부는 내가"** | 역할 분담 강조 |
| **"근거 있는 암기"** | 차별점 강조 (다른 앱에 없는 것) |
| **"5분 안에 시험 준비 시작"** | 속도/편의 강조 |

## 8. 벤치마킹 항목별 정리

### Decard가 반드시 구현해야 할 기능

| 우선순위 | 기능 | 참고 앱 | 이유 |
|---|---|---|---|
| 1 | **SRS (간격 반복)** | Anki(FSRS), Brainscape(CBR) | 3사 모두 보유. 없으면 "학습 앱"으로 인정 안 됨 |
| 2 | **학습 통계/진도** | Quizlet(Memory Score), Brainscape | 사용자 리텐션의 핵심 동기 부여 |
| 3 | **다양한 학습 모드** | Quizlet(Match/Test/Write) | 학습 방식 다양화 |
| 4 | **오프라인 학습** | Quizlet Plus, Brainscape Pro | 모바일 사용성 |
| 5 | **커뮤니티 덱 공유** | Anki(Shared Decks), Quizlet | 네트워크 효과 (후순위) |
